# StreamNumDeck v0.1.0 Beta

The first public beta of StreamNumDeck for Windows 11 x64.

## What's included

- Two independent assignment layers controlled by the physical NumLock state.
- Profiles with custom icons and switching from the navigation rail or system tray.
- OBS WebSocket 5.x actions for scenes, sources, inputs, streaming, recording, replay buffer, and media sources.
- Audio playback, microphone/system mute, master volume, and per-application volume controls.
- Program, file, folder, web-link, and multi-step keyboard macro actions.
- Global keyboard capture, emergency pause, tray controls, startup option, and draggable microphone-mute overlay.
- Automatic Windows-language selection with 22 included localizations.

The x64 MSIX is self-contained; .NET and Windows App SDK do not need to be installed separately.

## Installation

This beta is test-signed. Download all three release assets, verify the MSIX using `SHA256SUMS.txt`, then import `StreamNumDeck-Development-Signing.cer` into **Local Computer → Trusted People** before installing the MSIX. Administrator rights are required for the certificate import.

Expected certificate thumbprint:

```text
C340046FA11736792CC65D92F0B889A3FE2DCA85
```

Only trust the certificate downloaded from this official release. See the [bilingual installation guide](https://github.com/nikartom/StreamNumDeck/blob/v0.1.0/docs/install-msix.md) for exact steps.

Первый публичный beta-релиз StreamNumDeck для Windows 11 x64.

## Что входит в релиз

- Два независимых слоя назначений, переключаемые физическим состоянием NumLock.
- Профили с пользовательскими иконками и переключением на боковой панели или через системный трей.
- Действия OBS WebSocket 5.x для сцен, источников, входов, трансляции, записи, буфера повтора и медиаисточников.
- Воспроизведение звука, отключение микрофона и системного звука, общая громкость и громкость отдельных приложений.
- Запуск программ, открытие файлов, папок и ссылок, а также многошаговые клавиатурные макросы.
- Глобальный перехват клавиш, аварийная пауза, управление из трея, автозапуск и перетаскиваемый оверлей отключённого микрофона.
- Автоматический выбор системного языка Windows и 22 локализации.

MSIX x64 является автономным: отдельно устанавливать .NET и Windows App SDK не нужно.

## Установка

Эта beta-версия подписана тестовым сертификатом. Загрузите все три файла выпуска, проверьте MSIX по `SHA256SUMS.txt`, затем импортируйте `StreamNumDeck-Development-Signing.cer` в хранилище **Локальный компьютер → Доверенные лица** и установите MSIX. Для импорта сертификата нужны права администратора.

Ожидаемый отпечаток сертификата:

```text
C340046FA11736792CC65D92F0B889A3FE2DCA85
```

Доверяйте только сертификату, загруженному из этого официального выпуска. Точные команды приведены в [двуязычной инструкции](https://github.com/nikartom/StreamNumDeck/blob/v0.1.0/docs/install-msix.md).
